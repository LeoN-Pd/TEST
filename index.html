視差滾動效果

作者：Vineet Kumar

什麼是視差滾動？

視差滾動是一種網頁設計技巧，讓網站背景的移動速度比前景慢。當訪客向下捲動時，這會產生類似 3D 的深度效果，打造更具沉浸感的瀏覽體驗。視差效果源自視覺錯覺；人的雙眼會將靠近的物體看得比遠方更大，因此我們會覺得遠處的物體移動得較慢。這種錯覺長期被運用在各種媒介的視差呈現，營造出逼真的效果。最早的應用可以追溯到迪士尼的《白雪公主與七矮人》等傳統動畫，以及《超級瑪利歐》等電玩遊戲。隨著 CSS 與 HTML 的進步，視差效果也演化為今日熟悉的網頁設計技巧。

最早的瀏覽器視差滾動實作之一，是開發者 Glutnix 在 2007 年於部落格分享的案例，透過 JavaScript 與 CSS2 示範並支援當時的 IE6 等瀏覽器。不過直到 2011 年 HTML5 與 CSS3 普及後，這項技巧才在網頁設計圈廣受歡迎。支持者認為這是擁抱網頁流動性的簡單方法，許多人利用視差背景吸引使用者、提升整體體驗。然而普渡大學 2013 年的研究指出：「……儘管視差滾動強化了使用者體驗的某些面向，但不一定能全面提升整體體驗。」


為什麼要使用視差圖像？

    設計師應該嘗試視差圖像有兩個主要原因：

  • 幫助講故事

    視差圖像非常適合敘事。好的設計就是良好的溝通——當我們設計網站時，希望和訪客交流，並為產品、服務或品牌講述引人入勝的故事。視差滾動能讓訪客沉浸在獨特的線上體驗中，甚至可以打造單頁網站，讓使用者無須跳轉多個頁面，就能完整閱讀你的故事。

  • 提升用戶參與度

    視差圖像也能提升用戶參與度。降低網站跳出率是網頁設計的重要目標之一；跳出率代表只瀏覽單一頁面就按下「返回」或關閉分頁的訪客比例。高跳出率會影響搜尋排名，因此任何能降低跳出率的作法都值得嘗試。設計精良的視差頁面能帶來動態體驗，通常能讓使用者停留更久。


如果想在網站加入視差滾動效果，請留意以下建議。

  • 挑重點使用

    將視差滾動用在特定元素而非所有頁面，效果會更好。可以只放在頁首或標題，或僅限首頁。目標是讓視差效果強化網站設計，而非分散使用者注意力。

  • 影像壓縮

    視差滾動需要多種媒體檔與 CSS 形狀來營造深度感，但過大的圖片會拖慢載入，影響效能與體驗，記得先做好壓縮。

  • 善用色彩

    視差滾動不只有前景和背景的圖像，色彩同樣重要。背景可採較低飽和度的色調，前景則使用更活潑的顏色，以營造更強烈的深度層次。
